Overview:

Steve Chat was a group programming assignment that was written in Q2 2012. A simple IRC-clone, its purpose was to test our knowledge of networking and client-server synchronization in Java. I wrote the entire Client from scratch and contributed several methods to the Server code.

Feature List:

  • Fully functional server and client
  • Many commands that IRC users will be familiar with
  • (Unlimited) message history
  • Tab-completion
  • Logging
